# 🏔️ Trekking Data Analysis & Prediction App

This project is a group assignment developed as part of coursework. It is a machine learning-based web application built using Streamlit to analyze and predict trekking-related data in Nepal.

## 📌 Features

* Data exploration and visualization
* Machine learning prediction system
* Interactive web interface using Streamlit

## 🗂️ Project Structure

* `app.py` → Main application
* `explore_page.py` → Data visualization page
* `predict_page.py` → Prediction page
* `data/` → Dataset files
* `model/` → Saved ML model
* `notebook/` → Jupyter notebook

## ⚙️ Technologies Used

* Python
* Streamlit
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

## ▶️ How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the application:

```
streamlit run app.py
```
